# Sreeja Vijayan - Portfolio

A modern, responsive portfolio website showcasing my projects, skills, and professional experience.

## Features

- ✨ **Modern Design**: Clean and elegant UI with smooth animations
- 📱 **Fully Responsive**: Works seamlessly on desktop, tablet, and mobile devices
- 🎨 **Beautiful Gradients**: Eye-catching gradient backgrounds and text
- ⚡ **Smooth Scrolling**: Smooth navigation throughout the page
- 🎯 **Interactive Elements**: Hover effects, animations, and smooth transitions
- 📧 **Contact Form**: Built-in contact form for inquiries
- 🔗 **Social Links**: Easy links to GitHub, LinkedIn, Twitter, and email

## Sections

1. **Hero Section**: Eye-catching introduction with call-to-action buttons
2. **About Section**: Brief bio and key statistics
3. **Projects Section**: Showcase of featured projects with descriptions and technologies used
4. **Skills Section**: Organized list of technical skills by category
5. **Contact Section**: Contact form and social media links

## Technologies Used

- **HTML5**: Semantic markup structure
- **CSS3**: Modern styling with Flexbox and CSS Grid
- **JavaScript**: Vanilla JS for interactivity (no frameworks)
- **Google Fonts**: Poppins font family for modern typography

## Customization

To customize this portfolio for yourself:

1. **Update Personal Information**:
   - Replace "Sreeja Vijayan" with your name
   - Update the subtitle and bio in the About section
   - Modify project descriptions and links
   - Update skills list

2. **Update Projects**:
   - Replace project titles, descriptions, and tags
   - Add project links
   - Update gradient colors for project cards

3. **Update Contact Information**:
   - Replace social media links with your own
   - Update email address
   - Customize contact form handling

4. **Customize Colors**:
   - Edit CSS variables in `styles.css`:
     ```css
     --primary-color: #667eea;
     --secondary-color: #764ba2;
     ```

5. **Update Statistics**:
   - Modify the stats in the About section
   - Update technology counts

## Deployment

This site is set up to be deployed as a GitHub Pages site.

1. Push changes to the `main` branch
2. Enable GitHub Pages in repository settings
3. Your portfolio will be live at: `https://sreejavijayan17.github.io`

## Tips for Better Results

1. **Add Real Project Links**: Update project links to point to actual repositories or live demos
2. **Add Project Screenshots**: Replace placeholder gradients with actual project screenshots
3. **Set Up Form Backend**: Use a service like Formspree, Netlify Forms, or EmailJS for form submissions
4. **SEO Optimization**: Update meta tags and add Open Graph tags for better sharing
5. **Analytics**: Consider adding Google Analytics for tracking visitors
